What is an Armstrong Number?

An Armstrong number (also known as a narcissistic number) is a number that equals the sum of its own digits each raised to the power of the number of digits.

  • 153 is an Armstrong number: 13 + 53 + 33 = 1 + 125 + 27 = 153
  • 370 is an Armstrong number: 33 + 73 + 03 = 27 + 343 + 0 = 370
  • 9474 is an Armstrong number: 94 + 44 + 74 + 44 = 6561 + 256 + 2401 + 256 = 9474
